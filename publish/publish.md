# Publish

Useful information when publishing.

## Description:

EN
```
"Facebook AdBlock for your Timeline": Basically hides all ads injected by Facebook in your timeline. (This doesn't hide ads in other parts of the Facebook website, only on the timeline, for now)

This extension basically hide any content labeled as "Sponsored" or "Suggested post".

If you wish to propose new features, please open an issue on https://github.com/Vadorequest/fb-adblock-timeline/issues (easier to track down than comments)

If you wish to check the source code, here is the Github repository: https://github.com/Vadorequest/fb-adblock-timeline

If you wish to contribute, please check the CONTRIBUTING file in the repo.

IMPORTANT: Support for this extension is done in English and French only. 
This extension will only work with Facebook in French or English for now, it won't block ads in other language. 
If you wish to add new language please open an issue on Github: https://github.com/Vadorequest/fb-adblock-timeline/issues
```    


FR
```
"Facebook AdBlock pour votre fil d'actualité": Cache toutes les pubs de Facebook sur votre fil d'actualité. (Ne cache pas les pubs ailleurs sur le site, uniquement sur le fil d'actu, pour l'instant)

Cette extension cache tout content étant marqué comme "Sponsorisé" ou "Publication suggérée".

Si vous souhaitez proposer de nouvelles fonctionnalités, merci d'ouvrir un ticket sur https://github.com/Vadorequest/fb-adblock-timeline/issues (plus simple à suivre que les commentaires)

Si vous souhaitez consulter le code source, voici le dépôt Github: https://github.com/Vadorequest/fb-adblock-timeline

Si vous souhaitez contribuer, merci de d'abord consulter le fichier CONTRIBUTING sur le dépôt.

IMPORTANT: Le support pour cette extension est fait en Français et Anglais uniquement.
Cette extension ne fonctionnera qu'avec Facebook en Français ou en Anglais et ne bloquera pas les pubs dans d'autres langues.
Si vous souhaitez ajouter une nouvelle langue, merci d'ouvrir un ticket sur Github: https://github.com/Vadorequest/fb-adblock-timeline/issues
```
